## бҹЎ. бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһ•бҹ’бһҠбһҫбһҳ

### бһҸбһҫ Caelestia Shell бһҮбһ¶бһўбҹ’бһңбһё?

Caelestia Shell бһҮбһ¶ **desktop shell UI** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Hyprland (Wayland compositor) бһӣбһҫ Linuxбҹ” бһңбһ¶бһҹбһҡбһҹбҹҒбһҡбһҠбҹ„бһҷ QML (Qt) бһ“бһ·бһ„ C++бҹ” бһңбһ¶бһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһјбһ…бһҮбһ¶бҹ–

| бһҳбҹүбһјбһҢбһ»бһӣ | бһҜбһҖбһҹбһ¶бһҡ | бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ |
|---------|--------|-----------|
| Bar/Taskbar | `modules/bar/` | бһҡбһ”бһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ, workspaces, clock, tray |
| Launcher | `modules/launcher/` | бһ”бҹ’бһҡбһўбһ”бҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё, calculator, wallpaper |
| Lock Screen | `modules/lock/` | бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһ…бһ¶бһҖбҹӢбһҹбҹ„, password, fingerprint |
| Nexus (Settings) | `modules/nexus/` | бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ, audio, bluetooth, network |
| Dashboard | `modules/dashboard/` | бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„, media, performance |
| Notifications | `modules/notifications/` | бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ |
| OSD | `modules/osd/` | бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹҶбһЎбҹҒбһ„/бһ–бһ“бҹ’бһӣбһә |
| Sidebar | `modules/sidebar/` | бһҡбһ”бһ¶бһҡбһ…бҹҶбһ бҹҖбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ notifications |
| Session | `modules/session/` | бһ”бҹ’бһҡбһўбһ”бҹӢ shutdown/reboot/logout |
| Background | `modules/background/` | бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ, wallpaper, visualiser |
| Window Info | `modules/windowinfo/` | бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ„бҹ’бһўбһҪбһ… |
| Utilities | `modules/utilities/` | toasts, recording, quick toggles |

## бҹў. бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ Arch Linux + Hyprland + Caelestia

### бҹў.бҹЎ бһҸбҹ’бһҡбҹҖбһҳ USB Bootable

```bash
# Windows (PowerShell as Admin):
# 1. бһҠбҹ„бһҸ USB бһ…бһјбһӣ
# 2. бһҡбһҖбһҳбһҫбһӣ USB disk number:
Get-Disk

# 3. бһҹбһҡбһҹбҹҒбһҡ ISO бһ…бһјбһӣ USB (бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“ replace X бһҮбһ¶бһҳбһҪбһҷ disk number бһҡбһ”бһҹбҹӢ USB):
# бһ‘бһ¶бһүбһҷбһҖ Rufus: https://rufus.ie
# бһ¬бһ”бҹ’бһҡбһҫ command:
# diskpart
# list disk
# select disk X
# clean
# create partition primary
# format fs=fat32 quick
# active
# assign
# exit
# бһҡбһҪбһ… copy бһҜбһҖбһҹбһ¶бһҡ ISO бһ…бһјбһӣ USB (extract бһҠбҹ„бһҷ 7zip бһ¬бһҠбһ»бһҸбһҠбҹ„бһҷ Rufus)
```

### бҹў.бҹў бһҖбҹҶбһҺбһҸбҹӢ BIOS

1. Reboot вҶ’ бһ…бһ»бһ… F2/Del/F12 (бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“)
2. бһ”бһҫбһҖ **UEFI boot** (бһ”бһ·бһ‘ Legacy/CSM)
3. **Secure Boot** вҶ’ **Disable**
4. Boot order вҶ’ USB first
5. Save & Exit

### бҹў.бҹЈ бһҠбҹҶбһЎбһҫбһ„ Arch Linux

бһ–бҹҒбһӣ boot бһ…бһјбһӣ USB бһҡбһҪбһ… бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ command бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–

```bash
# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ internet
ping -c 3 google.com

# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ UEFI
ls /sys/firmware/efi/efivars   # бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“ files

# бһҳбһҫбһӣ partitions
lsblk
```

#### бһ…бҹӮбһҖ Partition (Manual)

```bash
# бһ”бһҫбһ…бһ„бҹӢбһҠбһ¶бһҖбҹӢ dual boot бһҮбһ¶бһҳбһҪбһҷ Windows:
# бһҷбһҫбһ„бһ”бһ¶бһ“ shrink D: бһ‘бһ»бһҖбһҡбһҪбһ…бһ бһҫбһҷ бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“ free space бһ‘бҹҶбһ“бҹҒбһҡ
fdisk /dev/nvme0n1    # бһ¬ /dev/sda
```

бһ’бҹ’бһңбһҫбһҸбһ¶бһҳбһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„ `fdisk`:

| Command | бһўбһҸбҹ’бһҗбһ“бҹҗбһҷ |
|---------|-----------|
| `g` | бһ”бһ„бҹ’бһҖбһҫбһҸ GPT partition table |
| `n` вҶ’ Enter вҶ’ `+1G` вҶ’ `t` вҶ’ `1` | EFI partition (1GB) вҶ’ type EFI |
| `n` вҶ’ Enter вҶ’ Enter | Root partition (free space бһ“бҹ…бһҹбһӣбҹӢ) |
| `w` | бһҹбһҡбһҹбҹҒбһҡбһҡбһҪбһ…бһ…бҹҒбһү |

```bash
# Format
mkfs.fat -F32 /dev/nvme0n1p1    # EFI
mkfs.ext4 /dev/nvme0n1p2         # root

# Mount
mount /dev/nvme0n1p2 /mnt
mount --mkdir /dev/nvme0n1p1 /mnt/boot

# бһ”бһҫбһҳбһ¶бһ“ Windows partition бһ…бһ„бҹӢ mount:
mkdir /mnt/windows
mount /dev/nvme0n1p3 /mnt/windows   # Windows partition
```

#### бһҠбҹҶбһЎбһҫбһ„ base system

```bash
pacstrap -K /mnt base linux linux-firmware base-devel git neovim networkmanager sudo cmake ninja

genfstab -U /mnt >> /mnt/etc/fstab

arch-chroot /mnt
```

#### бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

```bash
# Time
ln -sf /usr/share/zoneinfo/Asia/Phnom_Penh /etc/localtime
hwclock --systohc

# Language
echo "en_US.UTF-8 UTF-8" >> /etc/locale.gen
locale-gen
echo "LANG=en_US.UTF-8" > /etc/locale.conf

# Hostname
echo "arch" > /etc/hostname
cat > /etc/hosts <<EOF
127.0.0.1   localhost
::1         localhost
127.0.1.1   arch.localdomain arch
EOF

# Password
passwd    # бһҠбһ¶бһҖбҹӢ root password
useradd -mG wheel caelestia
passwd caelestia
echo "%wheel ALL=(ALL:ALL) ALL" >> /etc/sudoers

# Bootloader (systemd-boot)
bootctl install
cat > /boot/loader/loader.conf <<EOF
default arch.conf
timeout 4
console-mode max
editor no
EOF

# бһ”бһҫ dual boot бһҮбһ¶бһҳбһҪбһҷ Windows бһ”бһ“бҹ’бһҗбҹӮбһҳ:
cat > /boot/loader/entries/arch.conf <<EOF
title   Arch Linux
linux   /vmlinuz-linux
initrd  /initramfs-linux.img
options root=PARTUUID=$(blkid -s PARTUUID -o value /dev/nvme0n1p2) rw
EOF

cat > /boot/loader/entries/windows.conf <<EOF
title   Windows 11
efi     /EFI/Microsoft/Boot/bootmgfw.efi
EOF

# Services
systemctl enable NetworkManager systemd-resolved
```

### бҹў.бҹӨ бһҠбҹҶбһЎбһҫбһ„ Hyprland + Caelestia

```bash
# бһҖбҹ’бһ“бһ»бһ„ chroot:
pacman -Syu --noconfirm --needed \
  hyprland qt6-declarative qt6-base \
  ddcutil brightnessctl networkmanager lm_sensors fish pipewire wireplumber \
  libqalculate ttf-cascadia-code-nerd material-symbols-fonts noto-fonts \
  swappy aubio libcava

# AUR helper
cd /tmp
git clone https://aur.archlinux.org/yay-bin.git
cd yay-bin
makepkg -si --noconfirm --needed

# Caelestia
yay -S --noconfirm --needed quickshell-git caelestia-shell-git

# бһ…бһ¶бһҖбһ…бҹҒбһү
exit   # бһ…бһ¶бһҖбһ…бҹҒбһүбһ–бһё chroot
umount -R /mnt
reboot
```

```bash
# бһ”бһҫбһҖ terminal бһҖбҹ’бһ“бһ»бһ„ Hyprland (Super+Q)
cd ~/.config
git clone https://github.com/Mr-Reaz/Reaz.git quickshell/caelestia
cd quickshell/caelestia

# Build бһҳбҹ’бһҠбһ„бһҠбҹҶбһ”бһјбһ„

cmake --build build
sudo cmake --install build
sudo chown -R $USER ~/.config/quickshell/caelestia
```

### бһҹбҹҶбһ“бһҪбһҡ: бһҖбҹӮ QML бһ бһҫбһҷбһҳбһ·бһ“ reload?

бһҳбһҫбһӣ `shell.qml` бһҗбһ¶бһҳбһ¶бһ“ `settings.watchFiles: true` бһ¬бһ‘бҹҒбҹ” бһ”бһҫбһўбһҸбҹӢ бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бҹ”

### бһҹбҹҶбһ“бһҪбһҡ: бһҖбҹӮ C++ бһ бһҫбһҷбһҳбһ·бһ“бһғбһҫбһүбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣ?

бһҸбҹ’бһҡбһјбһң rebuild бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бҹ–
```bash
cd ~/.config/quickshell/caelestia
cmake --build build
sudo cmake --install build
caelestia shell -d
```

### бһҹбҹҶбһ“бһҪбһҡ: бһ…бһ„бҹӢ reset config бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ?

```bash
rm -rf ~/.config/caelestia
rm -rf ~/.config/quickshell/caelestia
```

бһҡбһҪбһ…бһҠбҹҶбһЎбһҫбһ„бһҗбҹ’бһҳбһёбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸбҹ”

### бһҹбҹҶбһ“бһҪбһҡ: бһ…бһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳ shortcut бһҗбҹ’бһҳбһё?

File: `modules/Shortcuts.qml`

```qml
CustomShortcut {
    sequence: "Super+Shift+N"
    onTriggered: {
        // бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбҹ’бһңбһёбһҳбһҪбһҷ
        print("Shortcut triggered!")
    }
}
```

### бһҹбҹҶбһ“бһҪбһҡ: бһ…бһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳ Widget бһҗбҹ’бһҳбһё?

1. бһ”бһ„бҹ’бһҖбһҫбһҸ file бһҗбҹ’бһҳбһёбһҖбҹ’бһ“бһ»бһ„ `modules/` (бһ§. `modules/mywidget/MyWidget.qml`)
2. Import бһңбһ¶бһҖбҹ’бһ“бһ»бһ„ `shell.qml`:

```qml
import "modules/mywidget"

ShellRoot {
    MyWidget {}
    // ... component бһ•бҹ’бһҹбҹҒбһ„бҹ—
}
```

### бһҹбҹҶбһ“бһҪбһҡ: бһ”бһҫбһҖбһҳбһ·бһ“бһҡбһҪбһ… / VM бһҳбһ·бһ“ boot?

- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ BIOS вҶ’ UEFI enabled, Secure Boot disabled
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ USB boot order
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ ISO file бһҒбһјбһ…бһ¬бһ‘бҹҒ (re-download)

## бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһ”бһүбҹ’бһ…бһ”бҹӢ

Caelestia Shell бһҮбһ¶ desktop shell бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“бһҒбҹ’бһ–бһҹбҹӢбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһҸбһ¶бһҳбһ…бһ·бһҸбҹ’бһҸ вҖ” бһ–бһёбһ–бһҺбҹҢ, font, layout, animation бһҡбһ бһјбһҸбһҠбһӣбҹӢ backend logicбҹ”

бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ…бһ¶бҹҶ:
- **QML** вҶ’ бһҖбҹӮбһҡбһҪбһ… save вҶ’ reload бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- **C++** вҶ’ бһҖбҹӮбһҡбһҪбһ… rebuild бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…
- **Config** вҶ’ бһҖбҹӮ `shell.json`
- **Git** вҶ’ commit бһ‘бһ»бһҖбһҮбһ¶бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·

бһҹбһјбһҳбһҹбҹҶбһҺбһ¶бһ„бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮ! рҹҡҖ
